### Setup

#### `.env` requirements

- `PORT` - Port Number
- `SECRET` - String for JWT tokens
- `DATABASE_URL` - database url to your postgres db

#### Running the app

- `npm start`
- Endpoint: `index.js`

#### Tests

- Unit Tests: `npm run test`
